# Algoritmos de Ordenação

## Descrição
Este repositório é o código-fonte do Trabalho Prático 2 da disciplina AEDS - CEFET/MG 2017/2. Algoritmos Implementados:
- Não eficientes:
  - Bubble Sort;
  - Insertion Sort;
  - Selection Sort;
- Eficientes:
  - Quick Sort;
  - Heap Sort;
  - Merge Sort;
